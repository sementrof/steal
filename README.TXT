1) Создать телеграм бота и запустить его, отправить ему любое сообщение (https://t.me/BotFather)
2) Скопировать токен бота (После создания бота, ботфазер выдаст токен)
3) Вставить токен между ковычек в файле main.py (TOKEN = "ВАШ ТОКЕН")
4) Получить чат айди своего телеграм аккаунта у бота (https://t.me/getidsbot)
5) Вставить чат айди между ковычек в файле main.py (CHAT_ID = "ВАШ ЧАТ АЙДИ")
6) Запустить Compile.bat
